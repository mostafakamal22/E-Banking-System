## Run Locally


Clone the project


```bash

  git clone https://github.com/mostafakamal22/E-Banking-System

```


Go to the project directory


```bash

  cd E-Banking-System

```


Install dependencies


```bash

  npm install

```

Start the server


```bash

  npm run start

```
